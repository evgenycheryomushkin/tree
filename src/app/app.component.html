Grow Tree project.


Three grows by itself. You can help growing the tree using mouse. On android you can use touch to grow tree. If you stop tree grow using "pause" button or pressing "space" on keyboard, then you can put leaves. Leaves can be put using mouse button.
You can change settings on settings button click. Please place issues to this Issue tracker Or send emails to cher888tree@gmail.com
